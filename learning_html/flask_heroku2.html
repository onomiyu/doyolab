herokuの初期設定とgithubに繋げる作業

2021/09/01

今までエラーが出てできなかったGitHubに連携する作業と、
herokuの初期設定もろもろを進める。

①仮想空間のアクティベート


source venv/bin/activate

②githubと連携する。

いままで出ていたエラー内容↓
gitのリポジトリにコミットしてもファイルが追加されない。
最初からステータスを見てリポジトリも作りなおして始める。
今回参考にしたページはこちらです。
1、今あるリポジトリを初期化してステータス確認。

git init
git status

2、ファイルをgitのインデックスに追加してローカルリポジトリにコミットする。

git add .
git commit -m "add new file"
そしてまた git status　で確認して黄色の部分のメッセージができれば連携成功。
自分のGitのページで確認しても、ファイルがしっかり更新されていたので、やっと連携成功。。。

③Heroku CLI　をインストールする

ターミナルでherokuを操作するために必要になる。

brew tap heroku/brew && brew install heroku

④herokuとGithubを連携させる。
